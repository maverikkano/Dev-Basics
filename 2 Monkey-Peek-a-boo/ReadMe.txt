CSS
===
1. display : flex  // flexible positioning of elements (parent needs to be flex)
flex-direction: column;  // relative placement- vertical or horizontal
justify-content: center;  // alignment- main-axis
align-items: center;  // aligment- cross-axis

2. user-select: none;  // text-selection flexibility

JavaScript
==========

2. EventListner  // adds event handler
